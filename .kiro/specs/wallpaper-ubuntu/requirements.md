# WallpaperUbuntu 需求文档

## 1. 项目概述

**项目名称**：WallpaperUbuntu  
**目标用户**：Ubuntu/Linux 桌面用户  
**核心目标**：提供一个简单、稳定、可自定义的本地壁纸自动切换工具

## 2. 功能需求

### 2.1 壁纸目录管理

**用户故事**: 作为用户，我希望能够添加和管理壁纸文件夹，以便组织我的壁纸收藏。

**验收标准**:
- [ ] 用户可以添加一个或多个本地文件夹
- [ ] 支持递归扫描子目录（可配置开关）
- [ ] 支持图片格式：jpg/jpeg, png, webp
- [ ] 无效文件自动跳过

### 2.2 自动切换壁纸

**用户故事**: 作为用户，我希望壁纸能够按照设定的时间间隔自动切换，提升桌面体验。

**验收标准**:
- [ ] 支持设置切换间隔（分钟/小时）
- [ ] 支持顺序播放模式
- [ ] 支持随机播放模式
- [ ] 支持不重复随机模式（循环一轮后重置）
- [ ] 后台定时执行，不阻塞系统

### 2.3 手动控制

**用户故事**: 作为用户，我希望能够手动控制壁纸切换，以便快速预览或选择喜欢的壁纸。

**验收标准**:
- [ ] 提供下一张壁纸按钮
- [ ] 提供上一张壁纸按钮
- [ ] 提供暂停自动切换功能
- [ ] 提供恢复自动切换功能

### 2.4 当前壁纸信息显示

**用户故事**: 作为用户，我希望看到当前壁纸的详细信息，以便了解壁纸来源。

**验收标准**:
- [ ] 显示文件名
- [ ] 显示文件路径
- [ ] 显示分辨率（可选）
- [ ] 显示所属文件夹

### 2.5 开机自启动

**用户故事**: 作为用户，我希望程序能够开机自动启动，无需每次手动运行。

**验收标准**:
- [ ] 支持启用/禁用开机启动
- [ ] 自动注册到 `~/.config/autostart/`
- [ ] 正确生成 .desktop 文件

### 2.6 系统托盘（可选）

**用户故事**: 作为用户，我希望通过系统托盘快速访问常用功能。

**验收标准**:
- [ ] 托盘图标显示
- [ ] 右键菜单包含：下一张、上一张、暂停/恢复、设置、退出
- [ ] 鼠标悬浮显示当前壁纸名称和切换时间

## 3. 设置项

| 设置项 | 描述 | 默认值 |
|--------|------|--------|
| 壁纸目录 | 多选文件夹列表 | 空 |
| 切换间隔 | 分钟/小时滑动选择 | 5分钟 |
| 切换模式 | 顺序/随机/不重复随机 | 顺序 |
| 开机启动 | 开/关 | 关 |
| 递归子目录 | 开/关 | 开 |

## 4. 非功能需求

### 4.1 性能
- 内存占用 < 100MB
- CPU 使用率低（定时任务驱动）
- 大目录扫描不阻塞UI

### 4.2 兼容性
- Ubuntu 20.04+
- GNOME 桌面（优先支持）
- Wayland 兼容

### 4.3 稳定性
- 壁纸路径失效时自动跳过
- 配置损坏时恢复默认
- 异常情况记录日志

## 5. 技术约束

### 5.1 技术栈
- .NET 8 (LTS)
- GtkSharp (原生GTK体验)
- System.Text.Json (配置存储)

### 5.2 壁纸设置方式
使用 gsettings 命令：
```bash
gsettings set org.gnome.desktop.background picture-uri "file:///path/to/image.jpg"
```

### 5.3 配置存储
- 路径：`~/.config/WallpaperUbuntu/config.json`
- 格式：JSON
- 原子写入保证

## 6. 正确性属性

### 6.1 壁纸切换正确性
- **属性**: 设置的壁纸必须是有效的图片文件
- **验证**: 文件存在且扩展名在支持列表中

### 6.2 切换顺序正确性
- **属性**: 顺序模式下，下一张索引 = (当前索引 + 1) % 总数
- **验证**: 边界条件正确处理

### 6.3 随机模式正确性
- **属性**: 不重复随机模式下，每轮循环内不重复
- **验证**: 已播放集合正确维护

### 6.4 配置持久化正确性
- **属性**: 配置保存后重启应用，设置保持一致
- **验证**: 序列化/反序列化正确

### 6.5 并发安全正确性
- **属性**: 定时切换不会与手动切换冲突
- **验证**: 使用锁机制防止重入
