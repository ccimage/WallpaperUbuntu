# 📄 Ubuntu 壁纸管理软件需求文档（PRD）

## 一、项目概述

**项目名称**：WallpaperUbuntu
**目标用户**：Ubuntu/Linux 桌面用户
**核心目标**：提供一个简单、稳定、可自定义的本地壁纸自动切换工具，提升桌面体验。

---

## 二、产品背景与问题

当前 Ubuntu（GNOME）自带壁纸功能存在以下问题：

* 自动切换功能较弱（配置复杂或不灵活）
* 不支持灵活的文件夹管理
* 无法细粒度控制切换规则（时间、顺序等）
* 缺乏轻量级第三方工具（或体验不佳）

👉 本项目旨在提供一个 **简单但功能完整** 的解决方案。

---

## 三、产品目标

### 核心目标

* 支持从本地文件夹读取壁纸
* 自动定时切换壁纸
* 提供简单直观的配置界面

### 次要目标

* 低资源占用
* 与 GNOME 深度兼容
* 易扩展（未来支持在线壁纸等）

---

## 四、功能需求

## 4.1 核心功能

### 1️⃣ 壁纸目录管理

* 用户可以添加一个或多个本地文件夹
* 支持递归扫描子目录（可选开关）
* 支持图片格式：

  * jpg / jpeg
  * png
  * webp

---

### 2️⃣ 自动切换壁纸

支持多种切换策略：

#### ⏱ 时间间隔切换

* 用户可设置：

* 分钟/小时/天 各自滑动选择
* 后台定时执行

#### 🔁 切换模式

* 顺序播放
* 随机播放
* 不重复随机（循环一轮后重置）

---

### 3️⃣ 手动控制

* 下一张壁纸
* 上一张壁纸
* 暂停自动切换
* 恢复自动切换

---

### 4️⃣ 当前壁纸信息显示

* 文件名
* 路径
* 分辨率（可选）
* 所属文件夹

---

### 5️⃣ 开机自启动

* 支持启用/禁用
* 自动注册到：

  * `~/.config/autostart/`

---

## 4.2 UI 功能


👉 系统托盘应用（Tray App）

* 托盘图标
* 右键菜单：

  * 下一张
  * 上一张
  * 暂停/恢复
  * 下一张名称 切换时间提示 鼠标悬浮后预览图片
  * 设置
  * 退出

---


## 4.3 设置项

| 设置项     | 描述    |
| ------- | ----- |
| 壁纸目录    | 多选    |
| 壁纸过滤    | 按图片分辨率，文件大小进行过滤        |
| 切换间隔    | 分钟/小时/天 各自滑动选择  |
| 切换模式    | 顺序/随机 |
| 开机启动    | 开/关   |
| 是否递归子目录 | 开/关   |

## 五、非功能需求

### 性能

* 内存占用 < 100MB
* CPU 使用率低（定时任务驱动）

### 兼容性

* Ubuntu 20.04+
* GNOME 桌面（优先支持）

### 稳定性

* 崩溃自动恢复（可选）
* 壁纸路径失效时自动跳过

## 六、技术方案

### 技术栈

| 语言和模块     | 优点            |
| ------ | ------------- |
| C# (.net 8) | 是我喜欢的 |
| GTKSharp   | 性能好 + 稳定      |


---

### 壁纸设置方式（GNOME）

使用 gsettings：

```bash
gsettings set org.gnome.desktop.background picture-uri "file:///path/to/image.jpg"
```

---

### 定时任务实现

可选方案：
* c#自带的定时器
* systemd timer

---

### 文件扫描

* 使用文件监听, 定时刷新（每10分钟）目录下的文件列表
* 启动时扫描 + 缓存列表

## 八、数据存储

配置文件：

```
~/.config/WallpaperUbuntu/config.json
```

示例：

```json
{
  "folders": ["/home/user/Pictures/wallpapers"],
  "interval": 300,
  "mode": "random",
  "recursive": true,
  "autostart": true
}
```
